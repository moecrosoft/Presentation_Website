# Presentation Website

This Streamlit website presents our analysis and findings on **HDB resale prices in Singapore**.  
It highlights **data exploration, feature engineering, and key insights** through interactive visualizations such as heatmaps, histograms, and bar charts.

## Requirements
- Python 3.8+
- Streamlit
- Pandas, NumPy
- Matplotlib, Seaborn
- Pillow

## Install dependencies 
- pip install -r requirements.txt

## How to Run
- streamlit run web.py
